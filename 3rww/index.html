3 Rivers Wet Weather Atlas

The 3 Rivers Wet Weather Atlas represents the next generation of mapping products from 3 Rivers Wet Weather. It consolidates previous geodata and maps into one geographic information system (GIS) hosted with ArcGIS Online.

The Atlas is currently in active development. This page lists all Atlas products and provides links to those that are live.

Visit the Atlases page for more information on the Atlas products. For information on data and map configuration products used in ArcGIS Online, visit the Maps & Services page.

Regional Green Infrastructure (RGI) Atlas Products

Status: version 1.0 online

These products replace the Rainways GI Map found at http://www.3riverswetweather.org/green/community-map

RGI Atlas (Editor)

The RGI Atlas (Editor) is an ArcGIS Online web application that provides full interactivity with RGI Operational Data in an attrative user interface (UI), including editing, location-based querying, and the creation of graphic data summaries using available data. As the data and data model is improved, additional querying and summary tools will be implemented.

URLhttp://alcogis.maps.arcgis.com/apps/webappviewer/index.html?id=ef7dc350ac004686a95c1b07054678d9

RGI Atlas (Public)

The RGI Atlas (Public) is an ArcGIS Online web application that provides interactivity with RGI Partner Data in an attrative user interface (UI). This includes location-based querying and the creation of graphic data summaries using available data. Editing is not implemented. As the data and data model is improved, additional querying and summary tools will be implemented.

URL:http://alcogis.maps.arcgis.com/apps/webappviewer/index.html?id=bd350e6dc31e42048a0a38ab91767117

Regional Sewer Infrastructure (RSI) Atlas Products

Status: version 1.0 available; ongoing improvements

These products replace the MDS OOM found at http://mds.3riverswetweather.org/login.aspx?src=WEBMAP&returnuri=http://www.landbasesystems.net/CollaborativeMaps_3rwwMDS.html

For a guide on how to use the RSI Atlas, visit the RSI Guide page.

RSI Atlas (Editor)

The RSI Atlas (Editor) is an ArcGIS Online web application that provides full interactivity with RSI Operational Data in an attrative user interface (UI), including editing, location-based querying, and the creation of graphic data summaries using available data. The implementation of additional analysis functionality, such as network tracing, is planned.

URL:http://alcogis.maps.arcgis.com/apps/webappviewer/index.html?id=d52b7d07014f43b88554a6a51a8abe20

Other Products

Several other products are planned to be moved to the 3 Rivers Wet Weather Atlas:

  • Planning and Engineering Tool: offline; new version planned
  • Homeowner's Tool: online; new version planned
