The Regional Sewer Infrastructre (RSI) Atlas User Guide

Note: this documentation is in active development

Interface

Map Controls

By default, drag the map to pan; use a scroll wheel if available to zoom.

Zoom buttons are located in the upper left-hand corner.

A Default extent button will reset the map to it's inital view.

A Locate me button will set the map to your location, if your browser and/or device has the necessary geo-location capabilities enabled (these vary by device).

Measurement Widget

Located in the top-right corner, the Measurement widget allows you to quickly measure distance and/or area on the map.

This tool can also quickly identify the longitude and latitude of a point placed on the map.

Printing Widget

Located in the top-right corner, the Printing widget allows you to print your current map view to a pdf or one of several image formats.

Editing Widget

Located in the top-right corner, the Editing widget at this time allows you to add annotation markers to the map and make notes.

In future iterations of this application, this widget will also enable direct editing of the sewer network.

Information Window

The left side of the screen contains a collapsible window that displays several types of information or options for configuring the map.

  • About: Provides an overview of the application.
  • Legend: Summarizes the symbols for features shown on the map currently.
  • Map Layers: All data layers available for viewing on the map, including RSI data.
  • Basemap Gallery: Available basemaps for display underneath the Map Layers.
  • Select Features (under 'more'): Provides the ability to filter features from the RSI data. See 'Analyzing' below for more information.
  • Summarize Features (under 'more'): Provides the ability to generate reports on the RSI data. See 'Analyzing' below for more information.

Common Tasks

Changing Map Layers

A number of data layers are available for viewing on the map in addition to the RSI data. This includes several map services provided by Allegheny County, which provide information ranging from parcel boundaries to recent aerial imagery. The user can opt to turn layers and sub-layers on or off as needed.

Changing Basemaps

The user can optionally select the base map shown underneath the map layers. By default, this is set to a global imagery map from Esri; however, other basemap options such as 'Dark Canvas' may provide the user with higher legibility and clarity for printing.
