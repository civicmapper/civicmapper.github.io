Regional Green Infrastructure (RGI) Atlas Products

RGI Atlas (Editor)

URLhttp://alcogis.maps.arcgis.com/apps/webappviewer/index.html?id=ef7dc350ac004686a95c1b07054678d9
TypeArcGIS Online - Web Application
RoleOperational/Editing
AccessRestricted via AlcoGIS Login & 3RWW Group

The RGI Atlas (Editor) is an ArcGIS Online web application that provides full interactivity with RGI Operational Data in an attrative user interface (UI), including editing, location-based querying, and the creation of graphic data summaries using available data. As the data and data model is improved (e.g., project cost), additional querying and summary tools will be implemented.

This product requires AlcoGIS Login & 3RWW Group access credentials to modify.

RGI Atlas (Public)

URL:http://alcogis.maps.arcgis.com/apps/webappviewer/index.html?id=bd350e6dc31e42048a0a38ab91767117
TypeArcGIS Online - Web Application
RolePublic Viewing
AccessPublic

The RGI Atlas (Public) is an ArcGIS Online web application that provides interactivity with RGI Partner Data in an attrative user interface (UI). This includes location-based querying and the creation of graphic data summaries using available data. Editing is not implemented. As the data and data model is improved (e.g., project cost), additional querying and summary tools will be implemented.

This product requires AlcoGIS Login & 3RWW Group access credentials to modify.
