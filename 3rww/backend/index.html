Maps and Services supporting the Regional Sewer Infrastructure (RSI) Atlas Products

These ArcGIS Online web map and service layers provide the configured map and geodata, respectively, that are used in the RSI Atlas products.

RSI Map (Editor)

URL:http://arcg.is/1PXweVa
TypeArcGIS Online - Web Map
RoleOperational/Editing
AccessRestricted via AlcoGIS Login & 3RWW Group

The RSI Map (Editor) is used to configure the availability and display of RSI Operational Data (Editor) geodata within the RSI Atlas (Editor) ArcGIS Online Web Application. Changes to map symbols, layouts, and other configuration settings in this map directly affect the display and functionality of the data in the RSI Atlas (Editor).

RSI Operational Data (Editor)

URL (point data): http://services1.arcgis.com/vdNDkVykv9vEWFX4/arcgis/rest/services/3RWWRSIFL/FeatureServer/0
URL (line data): http://services1.arcgis.com/vdNDkVykv9vEWFX4/arcgis/rest/services/3RWWRSIFL/FeatureServer/1
URL (tiles):http://tiles.arcgis.com/tiles/vdNDkVykv9vEWFX4/arcgis/rest/services/3RWWRSITL/MapServer
TypeArcGIS Online - Feature Layers and Tile Layer
RoleOperational/Editing
AccessRestricted via AlcoGIS Login & 3RWW Group

The RSI Operational Data (Editor) is the feature service referenced by the RSI Map (Editor) and RSI Atlas (Editor). Currently, this data is an import of the previous OneOverallMap (OOM) data. For the next iteration, it will be provided in using extended version of the Esri Local Government Information Model (LGIM) data model, which better supports geoprocessing and interoperability.

The RGI Map (Editor) is used to configure the availability and display of RGI Operational Data (Editor) geodata within the RGI Atlas (Editor) ArcGIS Online Web Application. RGI Operational Data (Editor)

For now, see RGI Partner Data (Public).

RGI Web Map (Public)

URLhttp://arcg.is/1PXtneX
TypeArcGIS Online - Web Map
RolePublic Viewing
AccessPublic

The RGI Map (Public) is used to configure the availability and display of RGI Partner Data (Public) geodata within the RGI Atlas (Public) ArcGIS Online Web Application; specifically, publicly visible attributes are limited via this product. Administrative changes to map symbols, layouts, and other configuration settings in this map directly affect the display and functionality of the data in the RGI Atlas (Public). This product requires AlcoGIS Login & 3RWW Group access credentials to modify.

RGI Partner Data (Public)

TypeArcGIS Online - Feature Layer
RolePublic Viewing
AccessPublic

The RGI Partner Data (Public) is the feature service referenced by the RGI Map (Public) and RSI Atlas (Public). It is a link to the data only; it requires AlcoGIS Login & 3RWW Group access credentials to modify.
