CivicMapper is a civic tech company that empowers civic organizations to utilize modern mapping technology for the greater good.

We believe that maps built on open data and open technology are a key ingredient to civic empowerment.

We design maps, curate geodata, and build spatial analysis tools for government, non-profits, and businesses operating in the civic sphere. We provide consulting services and build web applications that help civic organizations implement open mapping tools, ultimately enabling them and the public to use maps to make a difference.
